## ðŸ”¹ What is GitHub?

GitHub is a **cloud-based platform** used to host Git repositories.

- Branches

- Issue Tracking

## ðŸ§  Basic Git Concepts

- **Repository (Repo):** A project folder tracked by Git  
- **Commit:** A snapshot of changes  
  - Example: `uploading my project`
  - Example: `updating project`
- **Branch:** A parallel version of a repository  
- **Main / Master Branch:** Default branch  
- **Merge:** Combine branches  
- **Clone:** Download a remote repository  
- **Push / Pull:** Send and receive changes
